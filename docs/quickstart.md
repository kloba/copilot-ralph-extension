## Install

```bash
git clone https://github.com/kloba/copilot-ralph-extension
cd copilot-ralph-extension
./install.sh
```

## Arm a loop

```js
ap_loop({ prompt: "Refactor src/auth.js. Emit COMPLETE when done." })
```

Stop early:

```js
ap_stop({ reason: "changed plan" })
```
